# Path-to-GRC-Analyst

A structured, self-paced roadmap and notes repository for becoming a **Governance, Risk, and Compliance (GRC) Analyst**. This repo documents my learning journey — concepts, frameworks, tools, and certification prep — organized so it's useful both as personal notes and as a reference for anyone else studying GRC.


## 📌 About This Repo

I'm building this repo to:
- Track my progress toward becoming a GRC Analyst
- Keep organized notes on governance, risk, and compliance concepts
- Document frameworks (ISO 27001, NIST, SOC 2, COBIT, etc.)
- Log tools and platforms used in GRC roles
- Prepare for relevant certifications
